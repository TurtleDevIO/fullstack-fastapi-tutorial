# fullstack-fastapi-tutorial
Full-stack FastAPI tutorial with multiple frontend (SvelteKit, React) options. Learn to build a todo app with authentication, user data isolation, and type-safe API clients. Progressive branches for each tutorial.

## Tutorial Series

- [Tutorial 1: Project Setup & Tooling](https://turtledev.io/blog/fastapi-tutorial-1-project-setup-crud-api)
- [Tutorial 2: Building the Todo CRUD API](https://turtledev.io/blog/fastapi-tutorial-2-todo-crud-api)
- [Tutorial 3: SvelteKit SPA Setup with DaisyUI and Orval](https://turtledev.io/blog/fastapi-tutorial-3-sveltekit-spa-setup-orval)

## Branches

Each branch contains the **completed code** for that tutorial. To follow along with a tutorial, check out the previous branch as your starting point.

| Branch | Starting point for |
|--------|-------------------|
| `00-start` | Tutorial 1 — empty project |
| `01-backend-setup` | Tutorial 2 — end state of Tutorial 1 |
| `02-backend-crud` | Tutorial 3 — end state of Tutorial 2 |
| `03-frontend-setup` | Tutorial 4 — end state of Tutorial 3 |

Clone the repo and check out the branch for the tutorial you want to follow:

```sh
git clone https://github.com/TurtleDevIO/fullstack-fastapi-tutorial.git
cd fullstack-fastapi-tutorial
git checkout 00-start  # or 01-backend-setup, 02-backend-crud, etc.
```
